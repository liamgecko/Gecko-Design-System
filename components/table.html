Table

Standard Table

To ensure consistency with tables add the base class .table to any table, then extend with custom styles or our various included modifier classes.

All tables should also be wrapped in a .table-responsive div to ensure tables remain fully responsive.

Data Table

Use .table-data to any table that is data driven such as Events, Forms etc.

Columns which do not contain fluid data, such as checkboxes and buttons, should be fixed with and cells appended with a .cell-fixed class.
