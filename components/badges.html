Badges

Useage

Badges can be used in a variety of ways - namely when creating tags within the product or when using the multi-select function. They can also be used alongside headings to convey a new/beta feature.

About Badges

By default, badges are grey. They can also be coloured depending on the requirements of the badge.

Dismissable Badge

A dismissable badge would be used when the user wishes to remove a badge. For example, removing a tag or removing an option from a multi-select.

Dismissable

Filter Tags

When filtering, badges are used to display each chosen option. An AND/OR option is then shown if two or more are chosen.
