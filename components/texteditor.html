Text Editor

About Text Editors

A text editor is a textarea with added capabilities for use in various scenarios - mainly, when we allow the user to customise text, such as modifying font weights, styles etc.

Default Editor

The default rich text editor contains minimal editing capabilities. All icons should contain a tooltip to aid usability.

Editor with Controls

An editor with controls allows you to change the purpose of the editor or extend the editor. For example, upload an attachment or create a mail merge tag.
